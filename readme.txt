＜概要＞
日本酒の銘柄にはなかなか変換で出てこないものが多いのでMS IME向けの辞書を作ってみました。
現在は、蔵元と県名が説明の欄に出るようになっています。

現在 36件登録されています。

辞書データはBSDライセンスとします。ご自由にお使いください

＜内容＞
- sake.dic		MS IMEのシステム辞書です。このデータを辞書として登録してください
- sake_use.dic		MS IMEのユーザー辞書です。編集用のアップしています。
- sake.txt		MS IME形式のテキストファイルです。編集用にアップしています。

＜使い方＞
sake.dicを使っているIMEの辞書に登録してください。一度設定した後に更新する場合は
登録した辞書を解除した後にWindowsを再起動して、辞書データを更新します。
その後に再び、辞書を登録してください。
